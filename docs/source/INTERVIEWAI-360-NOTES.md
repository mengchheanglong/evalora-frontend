# InterviewAI 360 Source Notes

This folder contains extracted text and notes from the earlier detailed InterviewAI 360 SRS/PDF. Use it as supporting detail, not as the product name.

## What was retained for Evalora

- Modular assessment platform concept.
- AI interview chat.
- Live coding assessment.
- Work-style/behavioral assessment.
- Leadership scenario assessment.
- Communication roleplay simulation.
- Problem-solving case study.
- AI evaluation and scoring with evidence.
- Candidate report generation.
- Analytics dashboard.
- Interview history/session management.
- API groups for auth, templates, sessions, responses, AI, code, reports.
- Database entities for users, organizations, templates, modules, questions, sessions, responses, messages, submissions, evaluations, reports, and notes.
- Acceptance criteria, edge cases, risks, and month-by-month roadmap.

## What changed in the refined Evalora version

- Product name is Evalora.
- Stack is fixed to Next.js, NestJS, PostgreSQL, Prisma, JWT, Tailwind, Socket.IO, Monaco, Judge0/sandbox, and an AI API provider.
- Team responsibilities are adjusted to the four-member Evalora team.
- Claims are tightened: AI supports human reviewers and does not make final hiring decisions.
